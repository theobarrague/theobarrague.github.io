# ICI. METZ — site public

Site statique destiné à GitHub Pages et aligné sur la version 0.2 de l’application, disponible sur iOS et Android.

- `/` : présentation des cinq rubriques — actualités, agenda, piscines, parkings et parcs
- `/support/` : assistance et questions fréquentes
- `/confidentialite/` : politique de confidentialité couvrant Firebase Analytics, Cloud Firestore, Cloud Messaging et la localisation

Le site ne nécessite aucune compilation. GitHub Pages peut le publier directement depuis la branche `main`, à la racine du dépôt.
